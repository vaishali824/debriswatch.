\# 🛰️ DebrisWatch AI



> AI system that predicts collision between

> ISRO satellites and space debris

> 72 hours before it happens.



\## 🚨 The Problem

India has 50+ operational satellites worth

thousands of crores. Space debris is growing

every year. Current systems predict collision

only 24 hours ahead — too late to maneuver.



\## ⚡ My Solution

DebrisWatch AI predicts 72 hours ahead —

giving ISRO 3x more time to act.



\## 🛰️ Satellites Monitored

\- CARTOSAT-3     (Earth Imaging)

\- RESOURCESAT-2  (Agriculture Mapping)

\- RISAT-2BR1     (Radar Imaging)

\- OCEANSAT-3     (Ocean Monitoring)

\- GSAT-19        (Communication)



\## 🔥 Features

\- 72-hour closest approach prediction

\- Risk Score Engine (0-100)

\- Auto-refresh web dashboard

\- Alert system with auto-report

\- Historical risk tracking database

\- Real NASA Space-Track data



\## 🛠 Tech Stack

Python, Flask, SGP4, NumPy,

SQLite, Chart.js, NASA Space-Track API



\## 🚀 How To Run

pip install -r requirements.txt

python app.py

Open: http://127.0.0.1:5000



\## 📊 How Risk Score Works

Distance    → 40 points

Time Urgency → 25 points

Velocity    → 20 points

Orbit Type  → 15 points

Total       → 100 points



\## 🎯 ISRO Relevance

Directly extends ISRO Project NETRA

Monitors actual operational ISRO satellites
